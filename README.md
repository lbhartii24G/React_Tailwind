# Personal Portfolio Website

A modern and responsive website built with React and Tailwind CSS.

## Features

- Responsive Design
- Hero Section
- Featured Projects
- Services Section
- Client Feedback
- Our Impact
- Pricing Plans
- FAQ Section
- CTA Section
- Footer

## Tech Stack

- React.js
- Tailwind CSS
- Vite
- Swiper.js
- Lucide React
